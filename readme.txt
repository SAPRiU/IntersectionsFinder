Программа работает с СУБД MS SQL SERVER.
Чтобы начать работу, необходимо развернуть базу данных.
Для этого надо перместить файлы "SegmentsDB.mdf" и "SegmentsDB_log.ldf" в папку ..MSSQL\DATA и произвести присоединение базы данных.
Также необходимо указать строку подключения в файле App.config, если она отличается от дефолтной.
